# 🦜 Hyacinth Macaw — Tailwind CSS v4 Landing Page

A vibrant, responsive landing page dedicated to the world's largest flying parrot. Built to demonstrate modern frontend styling techniques and optimized asset delivery.

## 🔗 [Live Demo](https://olhakhodakivska.github.io/tailwind-intro/)

## 🛠️ Tech Stack
* **HTML5** (Semantic structure)
* **Tailwind CSS v4** (Modern JIT styling, CSS variables)
* **Vite 7** (Lightning-fast build tool)
* **GitHub Pages** (Deployment via `gh-pages`)

## ✨ Key Features
- **Modern UI:** Deep blue color palette with custom text shadows and hover effects.
- **Fully Responsive:** Grid-based gallery that adapts from mobile to desktop.
- **High Performance:** Minified assets and optimized image loading.
- **Smooth Navigation:** Anchor links with smooth scroll behavior.
